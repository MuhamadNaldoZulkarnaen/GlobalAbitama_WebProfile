# Global Abitama Web Profile

Company profile website for **PT Global Abitama**, a company specializing in professional building cleaning and maintenance services.
LINK : globalabitama.ct.ws
---

## 📌 Overview

This project is a company profile website designed to showcase PT Global Abitama’s services, credibility, and portfolio in a modern, informative, and user-friendly way.

### Main Focus
- Clean and well-organized landing page design  
- Clear and interactive navigation  
- Easy-to-understand information  
- Seamless user experience (UX)
